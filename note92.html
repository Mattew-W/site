Sheet Merger Tool

This web app helps merge student data from two sources:

  • An official student roster
  • Grades from a 3rd party system (e.g., Runestone)

The challenge is matching students when names don't perfectly match.

Workflow:

  1. Paste or open the data in the text areas (CSV format).
  2. Click "Parse" for each dataset. This will create Sheet objects.
  3. The app will attempt to match rows based on a key column (e.g., Student ID).
  4. Mismatches will be identified for manual matching.
  5. Users can select columns to merge from the 3rd party data into the roster.
  6. Finally, the merged data can be downloaded.
